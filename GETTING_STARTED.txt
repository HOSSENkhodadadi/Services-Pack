Example: Let's add an "Image Generator" service

1. Backend - Create API endpoint (backend/services/views.py):
    
    @csrf_exempt
    @require_http_methods(["POST"])
    def image_generator(request):
        data = json.loads(request.body)
        prompt = data.get('prompt', '')
        # Your image generation logic here
        image_url = "generated-image-url"
        return JsonResponse({'image_url': image_url, 'status': 'success'})

2. Backend - Register URL (backend/services/urls.py):
    
    path('image-generator/', views.image_generator, name='image_generator'),

3. Frontend - Create page (frontend/image-generator.html):
    
    <input id="promptInput" placeholder="Describe the image...">

    <img id="resultImage">

4. Frontend - Add JavaScript (frontend/image-generator.js):
    
    async function generateImage() {
        const prompt = document.getElementById('promptInput').value;
        const response = await fetch('http://localhost:8000/api/image-generator/', {
            method: 'POST',
            headers: {'Content-Type': 'application/json'},
            body: JSON.stringify({prompt})
        });
        const data = await response.json();
        document.getElementById('resultImage').src = data.image_url;
    }

COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "Could not connect to server"
Solution: Make sure Django is running (python manage.py runserver)

Issue: Django import errors
Solution: Activate virtual environment and install dependencies
    source venv/bin/activate
    pip install -r requirements.txt

Issue: CORS errors
Solution: Check that your frontend URL is in CORS_ALLOWED_ORIGINS
    (backend/config/settings.py)

Issue: Changes not showing
Solution: Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
